# 🌍 MaverickMoments – Dynamic Travel Blogging Site

**MaverickMoments** is a dynamic and responsive travel blogging web application built as part of an internship project. The platform is designed to showcase global travel experiences with a rich user interface, allowing users to explore blogs, categories, and locations interactively.

## 🔗Project Link

![View on Figma](https://www.figma.com/design/LyBEkbV3I9C7lG1qt31C1c/internship-project?node-id=0-1&t=Softat2uragiki8U-1) 

## 📌 Features

- ✈️ Dynamic landing page with featured travel blogs and posts  
- 📚 Category-wise blog listing (Beaches, Japan, Wildlife, etc.)  
- 📽 Embedded travel videos and image galleries  
- 📅 Blog articles organized by date and location  
- 📄 Individual blog post pages with detailed content  
- 📱 Responsive design for mobile and desktop screens  
- 🔍 Easy-to-navigate UI for readers to explore travel stories
